### 运行
1. `git clone --depth=1 https://github.com/cnscorpions/ionic-music-app.git`
2. cd ionic-music-app
3. `npm install -g ionic cordova` 全局安装ionic和cordova，使用npx和yarn同理
4. `yarn/npm install`
5. `ionic serve -o`

<br/>

### 构建(以Android平台为例)
1. `ionic cordova platform add android` 添加Android平台支持
